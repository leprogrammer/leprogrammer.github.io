BuzzWord

BuzzWord is an adaptation of the word game Boggle in which the user or player has to find words from a 4 by 4 grid. This project was created as part of the course CSE219. This was a semester long project which introduced me to the software design process and led to the creation of a Software Design Description created from a Software Requirement Specification before this project was started. The created Software Design Description can be found here.

Explicit Memory Allocator

This project was created as part of the course CSE320. It manages up to four pages of memory through an address-ordered explicit list with immediate coalescing. This was developed entirely in C.

Java File Reader

This is a simple file reader created using the MIPS assembly language to display a Java file using video memory and highlights keywords specific to the Java syntax. This was developed as part of the course CSE220.

Shell

A fully functional shell built entirely in C. This project was created to utilize my knowledge of processes, control flow, and signals.
